my goal for this directory is to make a web app, that will be served statically, so using only javascript. The user will be presented with a quiz, where they will need to choose one element from characters.md and one element from hotdrinks.md. The questions are stated as "You are at an ice skating rink with your friends and are taking a break. You go to the vending machine and line up. You are first in line and are presented with these options. What do you choose?" Then "Which race/species do you most closely identify with from Lord of the Rings?".

Use only html, javascript and css.
